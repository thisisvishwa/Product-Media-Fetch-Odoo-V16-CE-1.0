Product Media via URL's Module

Version: 1.0

Author: Vishwa G

https://thisis.com

Overview

This module is designed for Odoo 16 Community Edition and allows dynamic fetching of product images and extra media from external URLs to be displayed on the website without storing them in the Odoo database.

Module Details

  • Technical Name: product_media_urls
  • Version: 1.0
  • Author: Vishwa G
  • Website: https://thisis.com

Required Functionality

3.1 Dynamic Image Fetching

The module dynamically fetches product images and additional media from external URLs and displays them on the website.

3.2 Flexibility

It ensures flexibility in displaying product images and media from various external sources and supports different content types.

3.3 Easy Configuration

Administrators can easily configure external URLs for product images and media through the Odoo backend module interface.

3.4 Error Logging and Display

Logs error details if image/media fetching encounters an error and displays an error view within the Odoo Backend for quick issue resolution.

Developer Guidelines

All functionalities must be accessible and modifiable through the Odoo backend module interface, adhering to Odoo Custom Addons standards and architecture.

Documentation

This document serves as a guide for users and administrators for installing, configuring, and troubleshooting the module.
